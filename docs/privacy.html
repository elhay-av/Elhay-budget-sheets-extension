Privacy Policy

Effective Date: May 8, 2026

This Privacy Policy applies to the Elhay's Family Budget - for Google Sheet Chrome Extension ("the Extension"). We are committed to protecting your privacy and ensuring transparency about how your data is handled. By using the Extension, you agree to the practices outlined below.

Data Handling and Usage

To comply with the Google Chrome Web Store Developer Program Policies, we explicitly declare the following regarding user data:

  • No Data Collection: We do not collect, save, store, or hold any personal or sensitive user data on external servers. All authentication and data processing happens locally between your browser and the authorized third-party APIs.
  • No Data Selling: We do not sell or transfer user data to third parties, apart from the approved use cases strictly required for the Extension's core functionality.
  • Strict Purpose Limitation: We do not use or transfer user data for purposes that are unrelated to the Extension's single intended purpose.
  • No Creditworthiness Usage: We do not use or transfer user data to determine creditworthiness or for any lending purposes.

Local Storage

The Extension uses Chrome's local storage mechanism strictly to save your active session token and a generated unique device ID. This is required to maintain your active session with the Finanda API and prevent repeated login requests. This data never leaves your device except to authenticate directly with the secure Finanda API endpoints.

If you have any questions or concerns about this privacy policy, please contact the developer via the Chrome Web Store support tab.
